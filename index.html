Global Object

console is a global object. It can be accessed anywhere - you can always call console.log()

There are other global objects in JavaScript-
  • setTimeout()
  • clearTimeout()
  • setInterval()
  • clearInterval()

Modules

When Node.js is installed globally, it comes with a set of built-in modules:
  • "path"
  • "os
  • "https"
These modules can be accessed in a .js file by const pathModule = require("path"). You can see the whole module by console.log(pathModule)
