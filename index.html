Modules

When Node.js is installed globally, it comes with a set of built-in modules:
  • "path"
  • "os
  • "https"
These modules can be accessed in a .js file by const pathModule = require("path"). You can see the whole module by console.log(pathModule)

Node Package Manager (NPM)

Introduction

Node Package Manager (NPM) is a command line tool and registry of third party libraries that you can add to Node applications. When you install Node it comes with NPM

Package.json

Before you add any Node packages to an application, you need to create a package.json file. Running "npm init" from your project's home directory will walk you through creating a package.json file.

Live reloading with Node

Running node --watch [filename] from project home directory will create a live server that will automatically reload when code is changed

--watch does require to you refresh the page over and over to display new changes. However, it doesn't require you to restart the Node server which is the point of --watch in the first place
